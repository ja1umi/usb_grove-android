## Practice projects using MCP2221A USB-I2C Bridge

Here are some of my projects that use the USB-I2C Bridge Board, which was developed by __minna-no-Lab__ using the Microchip MCP2221A. I created some Android apps that work with this board.  These apps let you check out the features of this board, like GPIO, 10-bit ADC, and 5-bit DAC. They also let you connect I2C devices (no I2C-related apps have been uploaded yet, though).

The apps in this repository have been tested and verified to work with the Sharp Aquos Sense 4 Basic (Android 12) phone.

For more details about this board, please visit the following website.

http://minnanolab.net/product/pro_USB-I2C_BRIDGEBOARD-V2-BREAD/pro_USB-I2C_BRIDGEBOARD-V2-BREAD.html

You can buy this board at the MARUTSU online store.

https://www.marutsu.co.jp/pc/i/1360420/

I haven't tested this yet, but hobbyists outside Japan can use the Adafruit MCP2221A Breakout as a replacement if they change the wiring appropriately.
